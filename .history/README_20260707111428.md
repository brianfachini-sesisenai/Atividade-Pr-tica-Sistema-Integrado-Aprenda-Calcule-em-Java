# Atividade Prática: Sistema Integrado "Aprenda & Calcule" em Java

Este projeto é um console application em Java desenvolvido para integrar e demonstrar os principais conceitos de **Programação Estruturada** e **Programação Orientada a Objetos (POO)**. O sistema oferece duas funcionalidades principais: uma calculadora aritmética completa e um guia de conceitos de desenvolvimento interativo no estilo flashcards.

## 🚀 Funcionalidades

### 1. Menu Calculadora
Permite a realização de operações matemáticas básicas estruturadas por meio da classe de domínio `calculadora`.
* **Soma** (`+`)
* **Subtração** (`-`)
* **Multiplicação** (`*`)
* **Divisão** (`/`) — contendo validação de segurança para impedir a divisão por zero.
* **Resto da Divisão** (`%`) — cálculo de módulo também protegido contra divisão por zero.

### 2. Guia de Conceitos
Um guia interativo composto por 10 temas fundamentais para o aprendizado de programação, organizados na classe `guia_de_conceitos`:
1. Operadores Lógicos (`&&`, `||`, `!`)
2. Encapsulamento (Getters e Setters)
3. Construtores
4. Classes e Objetos
5. Herança
6. Polimorfismo
7. Estruturas de Repetição (`while`, `do-while`, `for`)
8. Estruturas de Decisão (`if-else`, `switch-case`)
9. Operadores Aritméticos
10. Métodos

---

## 🛠️ Fundamentos Técnicos Implementados

O projeto foi construído respeitando as exigências acadêmicas de desenvolvimento em Java:

* **Operadores**:
  * **Aritméticos**: `+`, `-`, `*`, `/`, `%`
  * **Relacionais**: `==`, `!=`, `>=`, `<=`
  * **Lógicos**: `&&`, `||`, `!`
* **Estruturas de Decisão**: Uso de blocos `if`, `else` e estruturas `switch-case` para menus e direções de fluxo.
* **Estruturas de Repetição**: Uso de loop `do-while` para o fluxo de navegação do menu principal.
* **Orientação a Objetos**:
  * **Encapsulamento**: Atributos `private` na classe `calculadora`, acessíveis apenas por meio de Getters e Setters públicos.
  * **Construtor**: Construtor personalizado para inicialização dinâmica dos objetos.
  * **Classes de Domínio**: Separação clara de responsabilidades entre as classes `main`, `calculadora` e `guia_de_conceitos`.

---

## 📂 Estrutura de Arquivos

* [main.java](file:///src/main.java): Ponto de entrada do programa que controla o fluxo do loop principal, interação com o teclado via `Scanner` e redirecionamento de opções.
* [calculadora.java](file:///src/calculadora.java): Classe de domínio contendo atributos privados para números, construtor, getters/setters e métodos matemáticos.
* [guia_de_conceitos.java](file:///src/guia_de_conceitos.java): Classe responsável pela interface interativa de consulta teórica.

---

## 💻 Como Rodar o Projeto

1. Certifique-se de ter o **JDK** (Java Development Kit) instalado em sua máquina.
2. Compile os arquivos fonte a partir da raiz do projeto:
   ```bash
   javac -d bin src/*.java
   ```
3. Execute o programa:
   ```bash
   java -cp bin main
   ```
